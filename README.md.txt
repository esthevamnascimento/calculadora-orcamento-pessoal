# Calculadora de Orçamento Pessoal 💰

![Python](https://img.shields.io/badge/python-3670A0?style=for-the-badge&logo=python&logoColor=ffdd54)
![License](https://img.shields.io/badge/license-MIT-blue)
![Status](https://img.shields.io/badge/status-active-success)

Uma aplicação em Python para ajudar no controle financeiro pessoal, permitindo registrar receitas, despesas e visualizar seu saldo atual.

## ✨ Funcionalidades

- 📈 Adicionar receitas com descrição e valor
- 📉 Registrar despesas com categorização
- 💰 Cálculo automático do saldo disponível
- 📊 Visualização de resumo financeiro
- 📑 Relatório detalhado de todas as movimentações
- 🖥️ Interface simples via terminal

## ⚙️ Pré-requisitos

- Python 3.8 ou superior
- Git (para clonar o repositório)

## 🚀 Como Usar

1. Clone o repositório:
```bash
git clone https://github.com/esthevamnascimento/calculadora-orcamento-pessoal.git
cd orcamento-pessoal
Execute a aplicação:


python main.py
Siga o menu interativo:


=== Calculadora de Orçamento Pessoal ===
1. Adicionar Receita
2. Adicionar Despesa
3. Ver Resumo
4. Gerar Relatório
5. Sair
